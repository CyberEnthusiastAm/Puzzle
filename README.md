# Circuit Shift Puzzle

A browser-playable sliding puzzle game. Rebuild the numbered circuit by moving tiles into the open socket.

## Play

Open `index.html` in a browser.

## Features

- 3x3, 4x4, and 5x5 boards
- Solvable random shuffles
- Move counter, timer, and best score per board size
- Undo, hint, and keyboard controls
